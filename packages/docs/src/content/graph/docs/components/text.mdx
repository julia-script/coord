# Text Component

`Text` component allows you to display a line of text at a specific position on a graph.
It's simply a wrapper around SVG text with support for `@coord/graph` api.

```jsx live
import { Graph, Text } from "@coord/graph";

export default function MyGraph() {
  return (
    <Graph width={"full"}>
      <Text position={[0, 0]} fontSize={20}>
        Hello!
      </Text>
    </Graph>
  );
}
```
